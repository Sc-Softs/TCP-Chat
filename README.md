# <p align="center">TCP Chat</p>

##### <p align="right">Powerd with Node.js</p>

## Build
+ Do not need to do anything ,just install dependences
    ```bash
    yarn # or npm install
    ```
## Run
+ Run follow command
    ```bash
    yarn start # or npm run start
    ```
## Debug
+ Run follow command
    ```bash
    yarn watch # or npm run watch
    ```
